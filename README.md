# Executive-Sales-Analysis-Dashboard-
This project provides a comprehensive analysis of sales performance using a dataset of 500+ orders. The goal was to transform raw sales data into actionable business insights through an interactive Excel dashboard, focusing on revenue trends, regional performance, and product categories.

Key Performance Indicators (KPIs)
The analysis tracks the following core metrics:

Total Net Sales: Over $1.96M in total revenue.

Order Volume: 500 unique orders processed.

Total Discounts: $155K+ applied across transactions.

Average Order Value (AOV): ~$3,932 per transaction.

Total Quantity: 2,724 items sold.

Key Insights & Features
Monthly Sales Trend: Identified seasonal fluctuations with a peak in June (over $400K).

Regional Performance: North region leads in sales, followed closely by West and East.

Product Category Breakdown: High demand in Clothing and Electronics, making up the bulk of the revenue.

Order Segmentation: Classified orders into High, Medium, and Low categories to understand customer spending patterns.

Interactive Dashboard: Built using Excel Pivot Tables and Slicers for dynamic filtering by Region, Product, and Salesperson.

Data Structure
The analysis was performed on a dataset containing:

Order Details: ID, Date, Quantity, Unit Price.

Customer Info: Name, Region.

Product Info: Category and Product Name.

Financials: Gross Sales, Discounts, and Net Sales calculations.

Tools Used
Microsoft Excel: Data Cleaning, Pivot Tables, and Dashboard Design.

Power Query: for data transformation.
